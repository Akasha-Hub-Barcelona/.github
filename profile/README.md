# Akasha Hub Barcelona

This repository hosts the default community profile and shared documentation for Akasha Hub Barcelona.

## Mission

Akasha Hub Barcelona exists to:

- Stimulate curiosity and social self-awareness.
- Encourage projects that promote a coherent, sustainable, respectful, and healthy environment.
- Organize in-person and online activities to exchange knowledge and perspectives.
- Explore social challenges, tools, and trends at local and global levels.
- Facilitate discussion spaces, self-coordination, and collaborative problem-solving.
- Provide tools for collaboration, self-management, and project launch.
- Build and maintain a shared knowledge infrastructure.

## Vision

Akasha Hub Barcelona aims to be a decentralized, self-managed local community reference point where people can:

- Meet and exchange ideas.
- Co-create innovative solutions.
- Transform social infrastructure toward collective coherence and sustainable coexistence.

## Values

- Continuous learning and diversity
- Playful creativity
- Commitment and gratitude
- Glocal linking (global + local)
- Interdependence and openness
- Integrity and effectiveness
- Intentional action
